## Recording Tips

### Before Recording
1. Run the demo data setup script to populate the app
2. Clear browser cache and cookies
3. Set browser zoom to 100%
4. Close unnecessary browser tabs
5. Disable browser extensions that might interfere
6. Use incognito/private mode for clean UI
7. Test audio levels

### During Recording
1. Use 1920x1080 resolution (Full HD)
2. Record at 30 fps minimum
3. Use smooth mouse movements
4. Pause briefly between scenes for editing
5. Record narration separately for better quality
6. Keep cursor movements purposeful

### Screen Recording Software Options
- **OBS Studio** (Free, professional)
- **Loom** (Easy, cloud-based)
- **Camtasia** (Paid, full-featured)
- **ScreenFlow** (Mac, professional)

### Post-Production
1. Add background music (subtle, non-distracting)
2. Add text overlays for key features
3. Add transitions between scenes
4. Color grade for consistency
5. Add Flo logo/branding
6. Export at 1080p, H.264 codec
